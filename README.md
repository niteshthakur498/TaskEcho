# TaskEcho

A monorepo containing the backend and frontend for the TaskEcho application.

## Project Structure

```
TaskEcho/
├── backend/     # Java backend (Spring Boot)
├── frontend/    # Node.js frontend (React / Next.js)
└── README.md
```

## Getting Started

### Prerequisites

- Java 21+
- Node.js 20+
- npm 10+

### Backend Setup

```bash
cd backend
# setup instructions go here
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
